Experiment 3: Grounded RAG

Goal: answer questions with explicit evidence and clear verdicts. The system should support answerable questions, refuse unsupported ones, and detect conflicts across facts and qualifiers.

Plain-language version: the system must show its evidence and admit when it does not know.

How this experiment works

  • Extract structured facts from text (using a deterministic extractor stub).
  • Store facts with qualifiers (time, version, scope).
  • Ask questions that require evidence chains and conflict checks.
  • Score precision, recall, and conflict detection.

How to read the metrics

  • Supported Precision/Recall = correct answers with evidence.
  • Refusal Accuracy = correctly refuses unsupported questions.
  • Conflict Precision/Recall = finds contradictions when they exist.

  • Extraction Consistency = same facts from repeated extractions.
